Privacy Policy

Crown Money Financial Toolkit Chrome Extension — Last updated: 30 March 2026

Summary: Crown Money Financial Toolkit does not collect, transmit, or store any personal data on external servers. All your data stays on your device.

1. Information We Collect

We do not collect any personal information. The Crown Money Financial Toolkit extension operates entirely on your local device. Specifically:

  • No analytics or tracking scripts are included
  • No personal data is collected, stored, or transmitted to CrownMoney.com.au or any third party
  • No cookies are set by this extension
  • No browsing history or web activity is accessed or monitored

2. Data Storage

All calculator data, preferences, and settings are stored locally on your device using your browser's local storage. This data:

  • Never leaves your device unless you explicitly export it
  • Is only accessible to this extension
  • Can be cleared at any time via the Settings page
  • Can be exported and imported as a JSON file for personal backup

3. AI Assistant & API Keys

The extension includes a free built-in financial assistant that works entirely offline with no API key required.

Users may optionally connect an external AI provider (OpenAI, Anthropic, or Google Gemini) by entering their own API key. When an API key is configured:

  • The API key is stored locally on your device only
  • API keys are never transmitted to CrownMoney.com.au or any party other than your chosen AI provider
  • Chat messages are sent directly from your browser to your chosen provider's API
  • API keys are excluded from data exports for security
  • You can disconnect and delete your API key at any time in Settings

Each AI provider has their own privacy policy governing how they handle API requests. We encourage you to review their respective policies.

4. Permissions

This extension requests the following Chrome permissions:

  • Host permissions (api.openai.com, api.anthropic.com, generativelanguage.googleapis.com) — Only used when you explicitly connect an AI provider with your own API key. No requests are made to these services unless you configure and use the AI assistant feature.

5. Third-Party Services

This extension does not integrate with any third-party services by default. The only external network requests occur when:

  • You have connected an AI provider and send a chat message (the message is sent to your chosen provider's API)

No advertising networks, analytics services, or tracking pixels are used.

6. Children's Privacy

This extension is a general-purpose financial calculator tool. It does not knowingly collect any information from anyone, including children under 13.

7. Changes to This Policy

If we update this privacy policy, the changes will be reflected here with an updated date. Significant changes will also be noted in the extension's update changelog on the Chrome Web Store.
